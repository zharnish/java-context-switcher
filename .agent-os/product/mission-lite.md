# Product Mission (Lite)

Java Context Switcher is a localhost web application that helps Java developers on Windows instantly switch between installed Java versions by clicking a button, automatically updating their configured environment variable (e.g. `JAVA_HOME`) via a background batch script.

It serves individual Java developers and build engineers who manage multiple JDK versions and need to context-switch quickly without manually editing system environment variables. Unlike CLI-based version managers, Java Context Switcher provides a visual, browser-based interface accessible at `localhost:3001` with zero infrastructure requirements — launched in seconds via `npx`, configured through a human-readable local file, and requiring no PATH manipulation or shell plugins.
