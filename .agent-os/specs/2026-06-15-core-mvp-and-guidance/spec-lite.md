# Spec Summary (Lite)

Build the complete working localhost web application for Java Context Switcher — a fully functional Express + TypeScript backend with a React + Bootstrap frontend where developers can add named Java version buttons, click them to run a Windows `setx` batch script that updates `JAVA_HOME` (or a user-configured env var), and have all configuration persisted in a local Markdown file. This phase also includes embedded Oracle download links and step-by-step setup instructions so new users can onboard without external documentation.
