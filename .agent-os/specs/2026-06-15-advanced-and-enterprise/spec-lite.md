# Spec Summary (Lite)

Extend Java Context Switcher with power-user and team-oriented capabilities: support for multiple simultaneous environment variable targets, config file path override, Markdown config import/export, per-version notes, drag-to-reorder buttons, a shareable team defaults config layer, a local audit log of all switch events, and cross-platform support (macOS/Linux via shell scripts). This phase transforms the tool from a personal Windows utility into a team-ready, cross-platform Java environment management solution.
