# java-context-switcher
A UI that allows the user to quickly switch between Java Versions they need to build their apps.

<!-- Modified by AI on 06/15/2026. Edit #1. -->

## Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- Multiple Java JDK installations on your machine

## First-Time Setup

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd java-context-switcher
   ```

2. Install all dependencies (backend + frontend):
   ```bash
   npm run install:all
   ```

## Running the App

Start both the backend and frontend in development mode with a single command:

```bash
npm run dev
```

- **Frontend** → [http://localhost:5173](http://localhost:5173)
- **Backend API** → [http://localhost:3001](http://localhost:3001)

## Running Tests

```bash
npm test
```

## Building for Production

```bash
npm run build
```
